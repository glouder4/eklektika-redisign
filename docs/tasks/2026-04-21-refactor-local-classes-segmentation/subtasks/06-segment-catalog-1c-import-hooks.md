## Цель подзадачи

Изолировать логику обработчика `OnSuccessCatalogImport1C` (`updateProperties`) и связанные константы ИБ из [`local/php_interface/init.php`](../../../local/php_interface/init.php) в отдельный сегмент каталога, не смешивая её с CRM и ценообразованием.

## Описание работ

1. Вынести функцию `updateProperties`, константу `IBLOCK_ID_1C` и вспомогательные функции секций (`actionSection`) в класс или файл сегмента под `local/classes/catalog/import/` (имя по соглашению команды).
2. Зарегистрировать событие `catalog`/`OnSuccessCatalogImport1C` из bootstrap этого сегмента (или из общего bootstrap после ST-08).
3. Сохранить идентичность маппинга свойств в `APPLICATION_TYPES` построчно для элементов ИБ.
