## 🔧 Дебаг

### Если не работает сохранение:
1. Открыть DevTools → Console
2. Проверить ошибки JavaScript
3. Открыть DevTools → Network
4. Проверить запрос к `ajax.php`
5. Проверить Response

### Если не загружается фото:
1. Проверить размер файла (<5MB)
2. Проверить тип файла (image/*)
3. Проверить права на папку `/upload/`
4. Проверить логи PHP

### Если ошибка sessid:
1. Проверить, что сессия активна
2. Проверить cookie в браузере
3. Очистить кеш и cookie
4. Проверить, что sessid передается в запросе
