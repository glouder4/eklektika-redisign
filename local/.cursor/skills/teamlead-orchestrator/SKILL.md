# /teamlead

Используй навык для оркестрации потока между Tech Lead и разработчиками.

## Команда вызова

`/teamlead <next_steps_from_techlead>`

## Что делать

1. Прочитать `Next steps for Team Lead`.
2. Разбить на независимые задачи для параллельной работы.
3. Сформировать четкие постановки для Developer Squad.
4. После выполнения провести аудит результатов.
5. Сформировать задачи на доработку при необходимости.
6. Передать итог и статус обратно Tech Lead.

## Обязательные артефакты

- Task plan for developers
- Parallelization map
- Audit findings
- Handoff to Tech Lead
