# /dev-squad

Используй навык для запуска волны разработки по задачам Team Lead.

## Команда вызова

`/dev-squad <task_bundle_from_teamlead>`

## Что делать

1. Взять пакет задач от Team Lead.
2. Выполнить независимые задачи параллельно.
3. Провести самопроверку каждого потока.
4. Вернуть Team Lead отчеты по каждому потоку.

## Обязательные артефакты

- Implemented
- Validation
- Known limitations
- Ready for Team Lead audit
