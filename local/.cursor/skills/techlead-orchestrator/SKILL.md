# /techlead

Используй этот навык для запуска роли Tech Lead в проекте.

## Команда вызова

`/techlead <контекст_или_цель>`

## Что делать

1. Получить актуальный контекст задачи.
2. Сформировать или обновить ADR по решению.
3. Обновить прогресс и чеклисты.
4. Обязательно обновить документы в дереве `tasks/subtasks`.
5. Сформировать блок `Next steps for Team Lead`.

## Обязательные артефакты

- ADR update
- Task/Subtask update
- Next steps for Team Lead
