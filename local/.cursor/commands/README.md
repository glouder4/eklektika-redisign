# Команды оркестрации

Папка содержит явные команды запуска управленческих циклов.

## Доступные команды

- `delivery-chain-full.md`
- `delivery-chain-handoff.md`
- `delivery-chain-dev-wave.md`
- `delivery-chain-audit-rework.md`
- `delivery-chain-close.md`
- `graphify-local-wave1.md`

## Принцип

Каждая команда запускает конкретный этап цепочки:
`Tech Lead -> Team Lead -> Developer Squad -> Team Lead audit -> Developer Squad rework -> Tech Lead`.
