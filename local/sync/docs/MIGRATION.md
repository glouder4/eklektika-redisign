# Миграция без простоя

1. **События:** `local/events/events.php` подключаются через `events/requires.php`; старый `classes/events.php` удалён — дублирования регистрации нет.
2. **ajax.php:** логика перенесена в `sync/from-crm/InboundGateway.php`; старые классы на месте, поведение сохранено.
3. **Секрет входа:** по умолчанию не задан — поведение как раньше. Для prod скопировать `config.local.php.example` → `config.local.php`, задать секрет и передавать токен с CRM.
4. **Дальнейший вынос** классов в `sync/to-crm/` / `sync/from-crm/` — поэтапно с алиасами `require_once` и без удаления старых путей до контрольного срока.
