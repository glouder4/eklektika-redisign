# Subtask 01: Контракт и документация регистрации

Родительская задача: [task.md](../task.md)

## Цель

Синхронизировать документацию контракта регистрации INN-first и B24-by-ID с фактическими изменениями в коде.

## Объем работ

- Проверить, что в `functional-contract.md` отражены:
  - локальный lookup по ИНН перед вызовами B24;
  - работа через `OS_COMPANY_B24_ID` как основной ключ связи;
  - запрет fallback на поиск B24 по ИНН при невалидной локальной связке;
  - правило single-writer для записи `OS_COMPANY_B24_ID`.
- Проверить согласованность с `channels.md`, `runbook.md`, `README.md`.
- Обновить `registration-checklist.md` под новый контракт.

## Входы

- Diff по файлам регистрации/интеграции из текущей ветки.
- Текущие документы `local/sync/docs/*`.

## Выходы

- Явная ссылка на канонический task-артефакт и текущий статус.

## Зависимости

## Риски

## Индивидуальные критерии готовности
