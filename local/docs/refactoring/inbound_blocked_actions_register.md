# Регистр временно блокируемых inbound ACTION (`inbound_disabled_actions`)

Политика настраивается в `local/modules/yomerch.b24.inbound/config.local.php` — массив `inbound_disabled_actions`.

## Текущее состояние (по умолчанию)

- **Заблокированных ACTION по умолчанию нет** — массив пуст или не задан.
- Изменение списка — только после согласования с продуктом и командой Bitrix24 (business-effects gate).

## Шаблон для согласования (заполняется при включении блокировок)

| ACTION | Owner (роль) | Причина / риск | Exit-условие (когда снять блок) |
|--------|--------------|------------------|----------------------------------|
| _пример_ | product-owner | ... | ... |

После согласования скопируйте имя ACTION в `inbound_disabled_actions` на стенде и зафиксируйте дату в коммите/задаче.
